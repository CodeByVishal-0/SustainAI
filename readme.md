# SustainAI

AI-powered Sustainability Assistant built using:

- LangChain
- ChromaDB
- Hugging Face Embeddings
- Groq Llama 3
- RAG Architecture

## Features

- Sustainability Knowledge Base
- SDG Document Retrieval
- Natural Language Question Answering
- Multi-Agent Architecture (Upcoming)
